# HR Departament — peviitor.ro

Acest repository centralizează proiectele și taskurile interne ale echipei HR din cadrul organizației **peviitor.ro**.  
Gestionăm activitatea prin metodologie **Agile**, folosind sprinturi și un board de proiect GitHub.

---

##  Structură repository
```
HR_departament/
├── docs/           # Politici, proceduri, template-uri interne
├── onboarding/     # Materiale pentru voluntari și angajați noi
├── internships/    # Documente și convenții de practică
├── workshops/      # Materiale pentru traininguri și sesiuni
└── README.md
```
## Project Board (Agile)

Taskurile echipei sunt organizate în sprinturi pe GitHub Projects:  
 [Vezi board-ul HR →](https://github.com/peviitor-ro/HR_departament/projects)

**Structura unui sprint:**
- Backlog — toate taskurile planificate
- In Progress — ce se lucrează activ
- In Review — taskuri în verificare
- Done — finalizate

---

## Cum contribui

1. **Creează un Issue** pentru orice task nou sau problemă identificată
2. Atribuie-i un **label** relevant (`recruitment`, `onboarding`, `training`, `admin`)
3. Adaugă issue-ul la **Project Board** și setează sprint-ul
4. Lucrează pe un **branch** separat dacă adaugi documente/fișiere
5. Deschide un **Pull Request** cu o scurtă descriere a modificărilor
6. Așteaptă review din partea HR Lead-ului înainte de merge
